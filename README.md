# Система автоматизации бронирования авиабилетов (WPF + MS SQL Server)

Клиентское desktop-приложение для пассажиров и администраторов авиакомпании, разработанное на платформе **.NET (WPF)** с использованием прямой работы с СУБД **MS SQL Server** через интерфейсы **ADO.NET** (архитектура Code-Behind).

---

## 🚀 Основной функционал

### Для пассажиров:
*   **Регистрация и авторизация**: Вход в личный кабинет, регистрация новых пользователей с проверкой формата Email по регулярным выражениям (**Regex**).
*   **Поиск рейсов**: Просмотр актуального расписания полетов с точным временем вылета/прилета, городов и цен.
*   **Динамическая карта мест**: Интерактивный выбор кресла в салоне самолета. Занятые места автоматически подгружаются из БД, подсвечиваются красным цветом и блокируются для клика.
*   **Маршрутные квитанции**: Автоматическая генерация электронного авиабилета в формате **PDF** на Рабочий стол пользователя при успешном бронировании.
*   **Личный кабинет**: Просмотр истории своих заказов и возможность мгновенной отмены брони.

### Для администраторов:
*   **Управление расписанием (CRUD)**: Полноценная панель для добавления, изменения и удаления рейсов.
*   **Защита данных**: Встроенная проверка внешних ключей (база защищена от падения, если указан несуществующий ID самолета).

## 🛠 Стек технологий

*   **Язык программирования:** C# (.NET 8.0 / .NET 9.0)
*   **Технология интерфейса:** WPF (Windows Presentation Foundation, XAML)
*   **База данных:** MS SQL Server (ADO.NET / `Microsoft.Data.SqlClient`)
*   **Генерация отчетов:** iTextSharp (LGPLv2.Core)
*   **Валидация данных:** Регулярные выражения (Regex)

## 📦 Пошаговое развертывание проекта

### 1. Настройка базы данных
1. Откройте **SQL Server Management Studio (SSMS)** или инструмент работы с данными в Visual Studio.
2. Скопируйте и выполните SQL-скрипт из файла `AirlineDB_Fixed.sql` (находится в репозитории).
3. Скрипт автоматически удалит старую версию БД (если она была), создаст таблицы, настроит связи, ограничения и зальет тестовые данные.

### 2. Настройка строки подключения в C#
В проекте WPF откройте файлы `LoginWindow.xaml.cs`, `PassengerWindow.xaml.cs` и `AdminWindow.xaml.cs`. Найдите следующую строку:
```csharp
private readonly string connectionString = @"Server=YOUR_SERVER_NAME;Database=AirlineDB;Trusted_Connection=True;TrustServerCertificate=True;";
```
Замените `YOUR_SERVER_NAME` на имя вашего локального экземпляра MS SQL Server (например, `.\SQLEXPRESS` или `(localdb)\MSSQLLocalDB`).

### 3. Запуск приложения
1. Откройте файл решения `.sln` в **Visual Studio**.
2. Соберите проект (`Ctrl + Shift + B`).
3. Запустите приложение (`F5`).
